# immoprice-frontend
In diesem Repository wird unsere Seite [immoprice.ch](https://immoprice.ch) entwickelt, versioniert und verwaltet. Die neuste Version des main Branches wird anhand von GitHub Pages unter der oben genannten URL zur Verfügung gestellt. 
### API
Um eine Preisvorhersage zu generieren, machen wir AJAX Abfragen an unser Backend [immoprice-backend](https://github.com/Immobilienrechner-Challenge/immoprice-backend).  Weiter Informationen sind unter der Dokumentationsseite des backend verfügbar.
